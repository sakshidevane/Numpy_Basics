# NumPy Practice Notebook

## Overview

This repository contains my **NumPy practice notebook** created as part of my Python learning journey. This repository contains my NumPy practice notebook covering arrays,indexing, slicing, reshaping, mathematical operations and matrix basics.The notebook focuses on understanding the basics and important operations of the NumPy library through hands-on examples.

NumPy is one of the most useful Python libraries for numerical computing and array-based operations. It is widely used in data science, machine learning, and scientific computing.

## File Included

* `numpy_practice.ipynb` – Jupyter Notebook containing practice examples and explanations.

## Topics Covered

The notebook includes practice on the following concepts:

### 1. Array Creation Methods

* Creating arrays using `np.array()`
* Using `arange()` for sequences
* Creating arrays with `zeros()` and `ones()`
* Using `linspace()` for evenly spaced values
* Generating random numbers with NumPy

### 2. Array Properties

* Shape of arrays
* Size of arrays
* Data types (`dtype`)

### 3. Basic Mathematical Operations

* Minimum and maximum values
* Sum and mean
* Standard deviation
* `argmax()` and `argmin()`

### 4. Reshaping Arrays

* Changing dimensions using `reshape()`

### 5. Indexing and Slicing

* Accessing single elements
* Slicing 1D arrays
* Accessing rows and columns in 2D arrays

### 6. Boolean Indexing

* Filtering even numbers
* Applying conditions on arrays

### 7. Array Operations

* Addition, subtraction, multiplication, division
* Floor division
* Power operations
* Broadcasting concepts

### 8. Copying Arrays

* Reference assignment
  n- Copy using `.copy()`
* Understanding data changes in arrays

### 9. Matrix Operations

* Element-wise multiplication
* Matrix multiplication using `@`
* Dot product
* Transpose of a matrix

### 10. Advanced Operations

* Vertical stacking (`vstack`)
* Horizontal stacking (`hstack`)
* Column stacking (`column_stack`)
* Splitting arrays using `hsplit()` and `vsplit()`
